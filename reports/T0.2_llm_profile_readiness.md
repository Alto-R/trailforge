## 唯一阻塞：无 LLM endpoint

环境无 `torch`(已装) 外的 LLM 库，也无 API key。要跑 T0.2 协议（20–30 用户提取 + split-half 一致性 + 人工合理性）需提供其一：

1. **API key**（Anthropic / OpenAI / DeepSeek / DashSCOPE）→ 写 `call_fn` 调云端，最省事，成本低（几百次调用）。
2. **本地模型**（有 GPU）→ `pip install transformers`，下载 Qwen2.5-7B-Instruct 等中文友好模型，`call_fn` 走本地。

接上任一，`extract()` 即可直接跑；u_LLM 拼到 [user_repr.py](../src/user_repr.py) 的 `u` 后，CF 用户塔输入从 5 维升到 ~23 维，并解锁消融实验 E5（无 LLM 画像）。

## 判断标准（计划 6.5/T0.2）

一致性中位相关 >0.6 + 合理性均分 >3.5 → 通过；<0.4 或 <2.5 → 放弃 u_LLM，仅用 u_cluster + onboarding。
